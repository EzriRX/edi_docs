EzriRx - EDI 820 Payment Order/Remittance Advice Documentation

The below documentation outlines the 820 EDI file that represents a Payment Order/Remittance Advice.

This document is sent to wholesalers when payment is processed on an EzriRx order. On EzriRx, this typically occurs when an order is fulfilled by the wholesaler, unless the order was placed using wholesaler-provided credit, in which case the payment is processed when the pharmacy pays their invoice.

820 Payment Order/Remittance Advice
