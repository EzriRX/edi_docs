EDI Transaction Workflow

Our EDI integration follows a standard pharmaceutical supply chain workflow to ensure seamless order processing and fulfillment:

  1. Product Catalog Setup (832) - Vendors send us their product catalog and pricing information so we can set up their inventory on EzriRx.
  2. Purchase Orders (850) - For every new order placed by pharmacies, we send vendors an electronic purchase order.
  3. Order Acknowledgment (855) - Vendors send back an acknowledgment to confirm the order details, pricing, and availability.
  4. Shipment Notification (856) - When orders are shipped, vendors notify us with tracking and shipment details, which we use to inform the pharmacy.
  5. Invoicing (810) - Vendors send electronic invoices for completed orders to facilitate payment processing.

Supported Communication Methods

We currently support EDI file exchange via the following methods:

  • FTP/SFTP - Secure file transfer protocol for batch EDI file processing.
  • AS2 (Applicability Statement 2) - Secure, real-time EDI message exchange over the internet.
