EzriRx - EDI 855 Purchase Order Acknowledgement Specifications

The below documentation outlines the 855 EDI file that represents a Purchase Order Acknowledgement.

As a wholesaler, you should supply an 855 once you receive an 850 to confirm acknowledgement of the Purchase Order and provide any pre-shipment amendments.

See below for some example 855 documents that handle our accepted scenarios:

1. Purchase Order Acknowledged w/ Detail

This is an example of a Purchase Order that was acknowledged with no changes required. When this type of 855 is provided, no changes are made to the order on EzriRx, and we simply store the acknowledgement status on our end.

Hover the highlighted elements to see more details:
